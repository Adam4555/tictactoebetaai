# Tic Tac Toe AI Mini App

This is a simple Tic Tac Toe web app with an AI bot and wallet connect feature. Designed to be deployed as a mini app on Warpcast or shared directly online.

## Features
- Play against a basic AI bot
- Connect Ethereum wallet (MetaMask)
- Simple and mobile-friendly design
